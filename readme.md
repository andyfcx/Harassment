 - CKIP: 切詞
 - XGBoost: 分類
 - gensim: Word2Vec
## 判斷流程參考
1. 先 filter 案由
    1. 可保留的：
        1. 妨害自由
        2. 妨害名譽
        3. 社會秩序維護法
        4. 性騷擾防治法
        5. 性別工作平等法
        6. 性別平等教育法
        7. 損害賠償
        8. 侵權行為損害賠償
        9. 違反保護令
        10. 家庭暴力罪
        11. 家庭暴力防治法
        12. 偽造文書
        13. 恐嚇
        14. 妨害電腦使用
        15. 解聘
        16. 退學
        17. 給付資遣費
        18. 個人資料保護法
            
    2. 可過濾掉的
        1. 妨害性自主
        2. 傷害
        3. 竊佔
        4. 強制猥褻
        5. 賭博
        6. 詐欺
        7. 竊盜
        8. 妨害公務
        9. 毒品危害防制條例
        10. 偽證
        11. 殺人
        12. 殺人未遂
        13. 違反政府採購法等
        14. 毀損
        15. 公共危險
            
2. 再 filter 事實以及理由，可能要強調哪些行為（關鍵字）被辨識出來時是要保留的，需要討論一下怎麼做才能符合需求
